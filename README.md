# Duplicating the Azure Policy 
1. Go to portal.azure.com
2. Select the subscription where you want to deploy the Azure Policy
3. Policies > Definitions > Search for "Allowed resource types"
4. If you want to customize it like renaming the policy, you need to duplicate the policy first.
5. Located the built-in definition "Allowed resources"
6. Select the policy and select Duplicate definition
7. Definition location is where the azure policy will be assigned. Choose the a subscription
- Determine the scope to which the policy can be assigned. It can be a management group or a subscription.
